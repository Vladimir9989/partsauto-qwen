# Развертывание проекта на Timeweb

## Варианты развертывания

### Вариант 1: Node.js хостинг (рекомендуется)
Timeweb предоставляет Node.js хостинг с поддержкой Express приложений.

**Шаги:**
1. Зайдите в панель управления Timeweb
2. Создайте новый Node.js проект
3. Загрузите файлы проекта (весь каталог `partsauto-site`)
4. Установите зависимости: `npm install --production`
5. Настройте переменные окружения в панели управления:
   - `PORT` (обычно Timeweb предоставляет свой порт, оставьте по умолчанию)
   - `NODE_ENV=production`
6. Укажите точку входа: `server/index.js`
7. Запустите приложение

### Вариант 2: VPS/VDS (полный контроль)
Если у вас есть виртуальный сервер на Timeweb:

**Шаги:**
1. Подключитесь к серверу по SSH
2. Установите Node.js и npm
3. Склонируйте или загрузите файлы проекта
4. Установите зависимости: `npm install --production`
5. Установите PM2 для управления процессом: `npm install -g pm2`
6. Запустите приложение с PM2:
   ```bash
   cd partsauto-site
   pm2 start ecosystem.config.js --env production
   pm2 save
   pm2 startup
   ```
7. Настройте Nginx/Apache как обратный прокси (опционально)

### Вариант 3: Статический хостинг + отдельный сервер
Если хотите разделить фронтенд и бэкенд:

1. Фронтенд (Vite build) загрузите в статический хостинг Timeweb
2. Бэкенд (Express сервер) разместите на Node.js хостинге или VPS
3. Настройте CORS и обновите URL API во фронтенде

## Настройка домена

1. В панели Timeweb добавьте домен к вашему проекту
2. Настройте DNS записи (A-запись на IP сервера или CNAME на хостинг)
3. Включите SSL сертификат (Let's Encrypt) для HTTPS

## Мониторинг и поддержка

- Для постоянной работы используйте PM2 или systemd службу
- Настройте логирование: `pm2 logs partsauto-server`
- Регулярно обновляйте зависимости: `npm update`
- Мониторинг нагрузки на сервер через панель Timeweb

## Переменные окружения

Создайте файл `.env` в корне проекта:

```
PORT=3001
NODE_ENV=production
XML_URL=https://ycf.partsauto.market/partsauto-feeds/avito_feeds/cb9901e562a4f410bd4f9bf80c21d094.xml
CACHE_DURATION=300000
```

## Тестирование

После развертывания проверьте:
1. Доступность сайта по домену
2. Работу API: `https://ваш-домен/api/products`
3. Загрузку карточек запчастей
4. Поиск и фильтрацию

## Устранение неполадок

1. **Сервер не запускается**: проверьте логи в панели Timeweb
2. **Ошибки API**: убедитесь, что XML URL доступен
3. **Проблемы с CORS**: настройте CORS в `server/index.js`
4. **Высокая нагрузка**: увеличьте кэширование, настройте пагинацию
