# 🚗 PartsAuto - Каталог автозапчастей

Современный веб-каталог автозапчастей с расширенным функционалом поиска, фильтрации и сравнения товаров.

## 🌟 Основные возможности

### ✅ Реализовано (Версия 1.1.0)

- **📱 Адаптивный дизайн** - работает на всех устройствах (desktop, tablet, mobile)
- **🔍 Умный поиск** - поиск по названию с debounce
- **🎯 Фильтрация** - по бренду, категории, диапазону цен
- **📊 Сортировка** - по цене, названию (А-Я, Я-А)
- **❤️ Избранное** - сохранение понравившихся товаров (localStorage)
- **⚖️ Сравнение** - сравнение до 4 товаров одновременно
- **🖼️ Детальный просмотр** - модальное окно с полной информацией и галереей
- **🚀 Оптимизация** - ленивая загрузка изображений, анимации
- **🔖 SEO** - meta-теги, Open Graph для соцсетей
- **📄 Пагинация** - удобная навигация по страницам
- **🎨 Современный UI** - плавные анимации, toast-уведомления

## 🚀 Быстрый старт

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

Запустите два терминала:

**Терминал 1 - Frontend (Vite dev server):**
```bash

```

**Терминал 2 - Backend (Express API server):**
```bash
npm run server
```

Приложение будет доступно по адресу: `http://localhost:5173`

```bash

```

### Запуск продакшен версии

```bash
npm run preview
```

## 🎯 Использование

### Поиск и фильтрация

1. **Поиск по названию**: Введите текст в поле "Поиск"
2. **Фильтр по бренду**: Выберите бренд из выпадающего списка
3. **Фильтр по категории**: Выберите категорию
4. **Фильтр по цене**: Укажите диапазон цен (от/до)
5. **Сортировка**: Выберите способ сортировки результатов

### Избранное ❤️

1. Нажмите на иконку сердца на карточке товара
2. Товар добавится в избранное (сохраняется в браузере)
3. Перейдите на страницу "Избранное" через кнопку в шапке
4. Счетчик показывает количество избранных товаров

### Сравнение ⚖️

1. Нажмите на иконку стрелок на карточке товара
2. Можно добавить до 4 товаров для сравнения
3. Перейдите на страницу "Сравнение" через кнопку в шапке
4. Просмотрите таблицу с характеристиками товаров

### Детальный просмотр 🔍

1. Кликните на изображение или название товара
2. Откроется модальное окно с:
   - Галереей всех изображений
   - Полной таблицей характеристик
   - Кнопками "В избранное" и "Сравнить"
   - Контактной информацией продавца

### Мобильная версия 📱

- Нажмите на иконку фильтра (☰) в шапке для открытия панели фильтров
- Свайпайте галерею изображений
- Все функции доступны на мобильных устройствах

## 📝 API Endpoints

### GET `/api/products`

Получение списка товаров с фильтрацией и пагинацией.

**Query параметры:**
- `search` - поиск по названию
- `brand` - фильтр по бренду
- `category` - фильтр по категории
- `priceMin` - минимальная цена
- `priceMax` - максимальная цена
- `page` - номер страницы (по умолчанию: 1)
- `limit` - количество товаров на странице (по умолчанию: 20)

**Пример запроса:**
```
GET /api/products?search=двигатель&brand=Toyota&page=1&limit=20
```

**Ответ:**
```json
{
  "products": [...],
  "total": 150,
  "page": 1,
  "totalPages": 8
}
```

## 🎨 Кастомизация

### Изменение цветовой схемы

Отредактируйте файл `src/index.css`:

```css
:root {
  --primary-color: #0d6efd;
  --secondary-color: #6c757d;
  --success-color: #198754;
  --danger-color: #dc3545;
}
```

### Изменение количества товаров на странице

В файле `src/App.jsx`:

```javascript
const ITEMS_PER_PAGE = 20 // Измените на нужное значение
```

### Настройка источника данных

В файле `server/index.js` измените URL XML фида:

```javascript
const XML_FEED_URL = 'ваш-url-фида.xml'
```

- Максимум 4 товара в сравнении
- Данные избранного и сравнения хранятся только в браузере (localStorage)
- Контактная информация продавца - заглушка (требует интеграции с реальными данными)
